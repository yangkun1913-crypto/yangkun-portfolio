## 关键禁忌

| ❌ 不要做 | ✅ 应该做 |
|-----------|----------|
| `encodeURI()` 处理中文路径 | 直接使用中文路径 |
| `file://` 协议打开 | `python3 -m http.server 8080` |
| Card 添加白色背景 | 保持 `transparent` |
| IntersectionObserver 懒加载 | setTimeout 分批加载（每批6张/200ms） |
| `loading="lazy"` 属性 | 已在分批加载中处理 |
| 文件名含空格 | 用下划线替代空格 |
| 直接 `img.src` 全量加载 | 分批加载避免连接池拥堵 |

## 图片加载机制（核心）
- 109张图分19批加载，每批6张，批间间隔200ms
- 原因：浏览器同域名并发HTTP连接限制 ~6个
- 实现：`loadAt()` 递归 + `setTimeout`，位于 renderGallery 末尾
- 已验证：线上 109/109 全部加载成功

## 图片数据格式
```js
{ url: '作品集-web/人像摄影/1.jpg', w: 15, h: 22, category: 'portrait', folder: '' }
```
- `w:h` = 实际像素宽高比（用于 `aspect-ratio` CSS 属性防止变形）
- `category`: `'artist'` | `'portrait'` | `'still'`
- `folder`: 仅艺人拍摄填写（`'刘旸'` / `'刘润铭'` / `'张踩铃'` / `'郭碧婷'` / `'陈妍希'`）
