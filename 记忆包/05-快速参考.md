## 最常见操作

### 用户说"我加了新图" →
1. 运行维护指南中的 Python 扫描脚本
2. 替换 `const photos = [...]` 数组
3. 验证

### 用户说"改样式" →
1. 直接修改 `<style>` 中的对应 CSS
2. `.card`、`.gallery-item`、`.splash-*`、`.lightbox-*`

### 用户说"改文字" →
1. 介绍页文字在 `<div class="splash-text">`（第421-428行）
2. 分类标签在 `<nav class="filter">`（第436-444行）

### 用户说"部署" →
1. 优先推荐 GitHub Pages
2. 已有 git 仓库（`git init` 完成），直接 push

## 关键禁忌

| ❌ 不要做 | ✅ 应该做 |
|-----------|----------|
| `encodeURI()` 处理中文路径 | 直接使用中文路径 |
| `file://` 协议打开 | `python3 -m http.server 8080` |
| Card 添加白色背景 | 保持 `transparent` |
| observer 定义在 renderGallery 之后 | observer 必须在前面 |
| 文件名含空格 | 用下划线替代空格 |

## 图片数据格式
```js
{ url: '作品集-web/人像摄影/1.jpg', w: 15, h: 22, category: 'portrait', folder: '' }
```
- `w:h` = 实际像素宽高比（用于 `aspect-ratio` CSS 属性防止变形）
- `category`: `'artist'` | `'portrait'` | `'still'`
- `folder`: 仅艺人拍摄填写（`'郭碧婷'` / `'陈妍希'` / `'刘润铭'`）
