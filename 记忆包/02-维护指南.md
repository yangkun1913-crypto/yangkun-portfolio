## 如何添加/删除/替换图片

### 步骤 1：处理图片
```bash
# 将新图片放入 作品集/ 对应子文件夹（原始图）
# 压缩新图片（宽度 1500px，JPEG 质量 80）
sips --resampleWidth 1500 原始图.jpg -s format jpeg -s formatOptions 80 --out 作品集-web/目标文件夹/图片名.jpg
```

### 步骤 2：更新 data 数组
运行以下 Python 脚本自动扫描所有图片并生成新的 `photos` 数据：

```bash
cd "/Users/xx/Desktop/个人网站"
python3 << 'PYEOF'
import os, subprocess

base = "作品集-web"
photos = []

def gcd(a, b):
    while b: a, b = b, a % b
    return a

def scan_dir(path, category, folder=""):
    for f in sorted(os.listdir(path)):
        if f.startswith('.') or not f.lower().endswith(('.jpg','.jpeg','.png','.webp')):
            continue
        filepath = os.path.join(path, f)
        rel = os.path.relpath(filepath, ".")
        result = subprocess.run(
            ["sips", "-g", "pixelWidth", "-g", "pixelHeight", filepath],
            capture_output=True, text=True
        )
        w = h = 1
        for line in result.stdout.split('\n'):
            if 'pixelWidth' in line: w = int(line.split(':')[1].strip())
            if 'pixelHeight' in line: h = int(line.split(':')[1].strip())
        g = gcd(w, h)
        rw, rh = w//g, h//g
        if rw * rh > 100:  # 过于精确的分数简化
            rw, rh = round(w/100), round(h/100)
            g2 = gcd(rw, rh)
            rw, rh = rw//g2, rh//g2
        photos.append((rel, rw, rh, category, folder))

# 扫描艺人拍摄子文件夹
for artist in sorted(os.listdir(os.path.join(base, "艺人拍摄"))):
    p = os.path.join(base, "艺人拍摄", artist)
    if os.path.isdir(p) and not artist.startswith('.'):
        scan_dir(p, "artist", artist)

# 扫描单层文件夹
scan_dir(os.path.join(base, "人像摄影"), "portrait")
scan_dir(os.path.join(base, "静物摄影"), "still")

# 输出统计
cats = {}
for p in photos: cats[p[3]] = cats.get(p[3], 0) + 1
print(f"总计 {len(photos)} 张：艺人 {cats.get('artist',0)} | 人像 {cats.get('portrait',0)} | 静物 {cats.get('still',0)}")
PYEOF
```

### 步骤 3：替换 index.html 中的 `photos` 数组
- 找到 `const photos = [` 到 `];` 之间的全部内容
- 用脚本生成的新数据替换
- 格式：`{ url: '作品集-web/分类/文件名.jpg', w: 宽, h: 高, category: '类型', folder: '子文件夹' }`

## 如何修改介绍页内容

直接编辑 `index.html` 中的 `<div class="splash-text">` 块：
- 名字、头衔、地区、服务、品牌、微信
- CSS 样式在 `.splash-*` 类中

## 如何修改分类

1. 在 `index.html` 的 `<nav class="filter">` 中添加/删除 `<button class="filter-btn">`
2. 在 `photos` 数组中为新分类的图片设置对应 `category` 值
3. **注意**：`category` 值 `'artist'` 有特殊逻辑——会自动扫描 `folder` 字段生成子分类

## 关键注意事项

### 文件名规范
- ✅ 使用下划线代替空格：`20240123_优可丝_look拍摄2102.jpg`
- ❌ 避免空格：`20240123 优可丝 look拍摄2102.jpg`
- 原因：空格会导致 URL 加载失败（旧版浏览器）

### 路径规则
- 所有图片路径相对于 `index.html`
- 格式：`作品集-web/分类/文件名.jpg`
- **不要用 `encodeURI()`**——现代浏览器 + HTTP 服务器完美支持中文路径

### 本地测试
- **不要**直接用 `file://` 协议打开 HTML（Chrome 安全限制会阻止图片加载）
- **必须**通过 HTTP 服务器访问（`python3 -m http.server 8080`）
- 双击 `预览网站.command` 即可自动启动服务器并打开浏览器
