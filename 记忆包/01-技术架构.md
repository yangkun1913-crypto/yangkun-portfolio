# 技术架构详解

## 核心原则

1. **单文件架构**：全部代码在 `index.html`（~800 行），CSS 在 `<style>`，JS 在 `<script>`
2. **零外部依赖**：无框架、无 CDN、无 npm 包
3. **纯静态**：任何 HTTP 服务器直接托管即可工作
4. **中文友好**：文件名含中文，现代浏览器+HTTP服务器完美支持（**不要用 `encodeURI()`**）

## CSS 架构

### 变量定义（`:root`）
```css
--bg: #f2f0eb;           /* 页面背景色 */
--card: #fff;            /* 卡片背景（已弃用，现为透明） */
--text: #5f584e;         /* 次要文字 */
--text-active: #1f1a14;  /* 主要/激活文字 */
--border: #e8e3da;       /* 边框色 */
--shadow: rgba(140,130,115,0.06); /* 阴影色（已弃用） */
```

### 瀑布流布局（关键代码）
```css
.gallery {
  column-count: 2;        /* 手机 */
  column-gap: 8px;
}
@media (min-width: 768px)  { .gallery { column-count: 3; } }  /* 平板 */
@media (min-width: 1200px) { .gallery { column-count: 5; } }  /* 桌面 */
@media (min-width: 1800px) { .gallery { column-count: 6; } }  /* 大屏 */
```

### 卡片防断行（关键！）
```css
.gallery-item {
  break-inside: avoid;   /* 防止 CSS columns 把卡片切成两半 */
  margin-bottom: 8px;
}
```

## JavaScript 架构

### 数据结构
```js
const photos = [
  {
    url: '作品集-web/人像摄影/1.jpg',  // 图片路径（相对于 index.html）
    w: 15, h: 22,                       // 实际像素宽高比（用于 aspect-ratio）
    category: 'portrait',               // 'artist' | 'portrait' | 'still'
    folder: ''                          // 子文件夹名（仅艺人拍摄时填写，如 '郭碧婷'）
  },
  // ...67条记录
];
```

### 关键函数
| 函数 | 作用 |
|------|------|
| `enterSite()` | 隐藏介绍页，显示作品集 |
| `returnToIntro()` | 返回介绍页（从「关于」按钮触发） |
| `renderGallery()` | 根据 `filteredPhotos` 重建整个画廊 DOM |
| `applyFilter()` | 根据 `currentCategory` + `currentFolder` 过滤并重新渲染 |
| `buildSubFilter(cat)` | 为艺人拍摄动态生成子分类按钮 |
| `openLightbox(index)` | 打开灯箱预览第 index 张图 |
| `prevImage()` / `nextImage()` | 灯箱前后切换 |

### 懒加载机制
```js
const observer = new IntersectionObserver((entries) => {
  entries.forEach(entry => {
    if (entry.isIntersecting) {
      img.src = img.dataset.src;       // 真正加载图片
      item.classList.add('visible');   // 渐入动画
      observer.unobserve(item);
    }
  });
}, { rootMargin: '300px' });           // 提前 300px 加载
```

### 初始化顺序（关键！避免 TDZ 错误）
1. `observer` 必须定义在 `renderGallery()` 调用之前
2. `renderGallery()` 中 `document.querySelectorAll('.gallery-item')` 会遍历刚创建的元素
3. 每个 `.gallery-item` 通过 `observer.observe(item)` 注册懒加载

## 响应式断点

| 断点 | 宽度 | 列数 | 卡片间距 |
|------|------|------|----------|
| 手机 | < 768px | 2 | 8px |
| 平板 | 768-1199px | 3 | 10px |
| 桌面 | 1200-1799px | 5 | 12px |
| 大屏 | ≥ 1800px | 6 | 14px |

## 介绍页（Splash）布局

```css
.splash-inner {
  display: flex;
  align-items: center;    /* 垂直居中 */
  gap: 80px;              /* 图与文间距 */
}
.splash-photo { width: 360px; height: 360px; }  /* 正方形大图 */
.splash-text { flex: 1; text-align: left; }      /* 文字左对齐 */
```
移动端（< 768px）自动切换为上下堆叠（`flex-direction: column`）。
