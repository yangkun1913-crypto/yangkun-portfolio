# 许洋堃个人摄影作品集网站 — 项目概览

## 基本信息

| 项目 | 内容 |
|------|------|
| 网站名称 | 许洋堃摄影作品集 |
| 所有者 | 许洋堃（商业摄影师，杭州，从业5年） |
| 用途 | 给甲方展示摄影作品的个人作品集网站 |
| 访问方式 | 本地 `localhost:8080`（开发）/ GitHub Pages（部署后公开） |
| 技术栈 | 纯静态 HTML + CSS + JS（单文件），零依赖 |
| 总大小 | ~40MB（109张压缩图片 + 1个HTML文件） |

## 项目结构

```

├── index.html              ← 唯一入口，包含全部 HTML/CSS/JS
├── 作品集-web/             ← 压缩后的网站用图（1500px 宽，JPEG 80%）
│   ├── avatar.jpg          ← 许洋堃头像（圆形展示在介绍页）
│   ├── 艺人拍摄/
│   │   ├── 刘旸/     (3张)  ← NEW
│   │   ├── 刘润铭/   (3张)
│   │   ├── 张踩铃/   (3张)  ← NEW
│   │   ├── 郭碧婷/   (3张)
│   │   └── 陈妍希/   (3张)
│   ├── 人像摄影/  (72张)
│   └── 静物摄影/  (22张)
├── 作品集/                 ← 原始高清图（不上传Git，仅本地保留）
├── 预览网站.command        ← macOS双击启动本地服务器的脚本
├── .gitignore              ← 忽略 作品集/、.DS_Store 等
├── .nojekyll               ← 禁用Jekyll，加速GitHub Pages部署
└── 记忆包/                 ← 本项目记忆文档
```

## 设计风格

- **配色**：莫兰迪浅色系
  - 背景 `#f2f0eb`（暖灰白）
  - 文字 `#5f584e`（灰棕）
  - 强调文字 `#1f1a14`（深棕）
- **布局**：Pinterest 风格 CSS 瀑布流（`column-count`）
  - 手机 2 列 → 平板 3 列 → 桌面 5 列 → 大屏 6 列
- **介绍页**：极简杂志风，左图右文，纯白背景
- **卡片**：无白框，图片直接展示，圆角 8px，hover 放大 1.05x
- **灯箱**：无白框，`backdrop-filter: blur(8px)` 毛玻璃背景

## 页面结构

```
[介绍页 Splash]  →  [作品集 Gallery]
    ├─ 左：头像          ├─ 顶栏：分类筛选（全部/艺人拍摄/人像拍摄/静物拍摄）
    ├─ 右：名字+信息      │   ├─ 左上角「关于」→ 返回介绍页
    └─ 底：进入作品集 →   │   └─ 艺人拍摄子分类（郭碧婷/陈妍希/刘润铭）
                        ├─ 瀑布流图片
                        └─ 灯箱预览（← → 切换、Esc 关闭、触摸滑动）
```
